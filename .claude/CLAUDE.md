# Project Spec: SwiftDSA

A swift project for my personal DSA practice for interview rounds. Concepts are written as files. For example, for the MinHeap concept, we have 3 files:

- `MinHeap.swift` - I write my answers here
- `MinHeapAnswers.swift` - Final answers to refer too if I forget
- `MinHeapTests.swift` - Tests to run to verify my answer. Tested against non-answer file.

## Tech Stack

- Swift + Swift Testing

## Guidelines

- DO NOT build the app, it will generate `.build` artifacts. Run tests only if needed.
